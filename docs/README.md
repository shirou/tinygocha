## 🔧 技術スタック

### 主要技術
- **言語**: Go 1.24
- **ゲームエンジン**: Ebitengine v2.8.8
- **設定形式**: TOML
- **フォント**: MPlus1p + カスタム対応

### アーキテクチャパターン
- レイヤードアーキテクチャ
- Strategy Pattern
- Factory Pattern
- Observer Pattern
- State Pattern
